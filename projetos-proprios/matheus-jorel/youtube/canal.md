# Canal Matheus Jorel — DNA e Critérios de Conteúdo

Arquivo de referência para calibrar o que é uma boa sacada pro canal. Atualizado com base na análise de 100+ vídeos e seus dados reais de performance.

---

## Posicionamento do canal

**Para quem:** Infoprodutores, experts e prestadores de serviço que vendem (ou querem vender) via webinar e lançamento pago.

**Promessa central:** Aprenda a montar e escalar um webinar lucrativo — com ferramentas reais, estratégias testadas e bastidores de operações que faturam.

**Produto principal:** Checklist do Webinar Lucrativo

**Tom:** Direto, prático, sem guru. Fala de quem já fez, não de quem só leu a respeito.

---

## O que funciona (padrões dos vídeos que mais performaram)

### 1. ChatGPT / IA aplicados a marketing
O maior multiplicador de views do canal. Qualquer tema combinado com IA performa muito acima da média.
- **Melhor exemplo:** "O Melhor Prompt do ChatGPT Para Fazer Copywriting" — 6.493 views (outlier absoluto)
- Outros: "Comandos ChatGPT para Copywriting" (567), "Como Fazer Copy com ChatGPT" (524)
- **Regra:** Se o tema puder ter um ângulo de IA, sempre priorizar esse ângulo.

### 2. Tutoriais de ferramentas específicas
Conteúdo que mostra COMO USAR uma ferramenta real performa consistentemente bem.
- "Como Fazer Webinar no Zoom" — 1.382 views
- "Como Fazer Webinar no YouTube com StreamYard" — 1.105 views
- "Melhores Ferramentas Para Webinar" — 606 views
- "Desvendando o Segredo do Sellflux" — 241 views
- **Regra:** Ferramentas que a audiência já usa ou quer usar = conteúdo de alto valor de busca.

### 3. Resultados reais com números concretos
Números específicos geram clique. Bastidores reais de operação são o diferencial do canal.
- "Como eu Fiz R$428k em 5 Meses" — 985 views
- "Funil de 8 Dígitos com Webinar: Estudo de Caso" — 309 views
- **Regra:** Sempre que tiver um número real (resultado de cliente, métrica de campanha), esse é o ângulo do vídeo.

### 4. Referências a figuras de autoridade reconhecidas
Associar o conteúdo a um nome que a audiência respeita amplifica o alcance.
- "Webinar Perfeito - O Script do Russell Brunson" — 2.590 views (segundo maior do canal)
- "10 Segredos de Russell Brunson Para Webinars Milionários" — 219 views
- "5 Estratégias de Marketing de Russell Brunson" — 265 views
- **Regra:** Russell Brunson, Alex Hormozi e outros referências do mercado americano ressoam bem.

### 5. Webinar + fundamentos
Vídeos que ensinam os fundamentos do webinar com profundidade performam bem — a audiência ainda está aprendendo.
- "Webinar Perfeito - Script Russell Brunson" — 2.590
- "Estrutura Completa Para um Webinar Lucrativo" — 590
- "Webinar - O Que é Webinar" — 565
- "Webinar - Como Fazer Webinar" — 506
- **Regra:** O "o que é webinar" funciona porque tem volume de busca. Fundamentos bem executados têm audiência garantida.

### 6. Comparativos e decisões binárias
"X ou Y — qual é melhor?" resolve uma dúvida real que a audiência já tem.
- "Webinar Ao Vivo ou Gravado: Qual é Melhor" — 507
- "Webinar ou Lançamento - Qual é o Melhor" — 206
- "Funil de Webinar Ainda Vale a Pena" — 380
- **Regra:** Sempre que o tema envolver uma escolha que o público já se faz, o comparativo é um ângulo forte.

### 7. Tendências com prazo específico
- "Lançamento Pago - A Tendência Para 2026" — 159 views (recente, ainda crescendo)
- **Regra:** Conteúdo de tendência com ano no título capta quem está pesquisando ativamente.

## O que não funciona (padrões dos piores performers)

### Copywriting voltado para carreira
A audiência do canal quer USAR copy pra vender — não se tornar copywriter. Conteúdo de "como ganhar dinheiro como copywriter" não conecta.
- "Princípios Para Ganhar Mais de 10k Como Copywriter" — 52 views
- "Como Ganhar 10k Rápido Como Copywriter" — 37 views
- "5 Sinais Que Você Nasceu Para Ser Copywriter" — 31 views

### "O que é X" para conceitos básicos de marketing
Funciona pra webinar (tem volume de busca), mas não pra conceitos genéricos.
- "Funil de Vendas - O Que é Funil de Vendas" — 14 views
- "Página de Captura - O Que é Página de Captura" — 30 views
- "Landing Page - O Que é Landing Page" — 45 views

### Dicas genéricas sem ancoragem em resultado ou ferramenta
- "10 Dicas Para Copywriters" — 36 views
- "5 Princípios Para Copywriters de Alex Hormozi" — 45 views
- "Como Quebrar as 5 Maiores Objeções" — 27 views

### Equipamentos e bastidores pessoais
- "Meus Equipamentos Para Trabalhar Como Copywriter" — 32 views

### Conteúdo muito sazonal ou nichado demais
- "Webinar - Não Cometa Esses Erros Na Black Friday" — 30 views
- "Como Configurar Domínio No ClickFunnels" — 55 views

## Fórmulas de título que funcionam

1. `ChatGPT — [ação específica] Para [resultado]`
2. `Como Fazer [X] com [Ferramenta conhecida]`
3. `Como eu Fiz [Número real] em [Tempo]`
4. `[Figura de autoridade] — [Técnica/Script/Método]`
5. `Webinar — [Tema específico da operação]`
6. `[X] vs [Y] — Qual é Melhor`
7. `[Tema] Para [Ano]`
8. `Funil de [X dígitos] com Webinar — Estudo de Caso`

## Critérios para uma boa sacada

Uma sacada vale virar vídeo se ela se encaixa em pelo menos um desses critérios:

✅ Vem de experiência real numa operação (não é teórica)
✅ Tem número concreto que pode entrar no título
✅ Envolve uma ferramenta que a audiência usa (Zoom, StreamYard, Sellflux, ActiveCampaign, etc.)
✅ Pode ser combinada com IA/ChatGPT de forma natural
✅ Responde uma dúvida binária que infoprodutores têm ("devo fazer X ou Y?")
✅ Conecta com webinar, lançamento pago ou funil de vendas
✅ É algo que clientes perguntam recorrentemente
✅ Vem de bastidores de uma operação que fatura (resultado real = credibilidade)

❌ Não vale se for dica genérica que qualquer canal de marketing poderia dar
❌ Não vale se for voltado pra carreira de copywriter
❌ Não vale se não tiver conexão com venda via webinar ou lançamento
❌ Não vale se for teoria sem resultado ou ferramenta concreta

## Temas que ainda não foram explorados (oportunidades)

Com base na análise do canal, faltam vídeos sobre:
- WhatsApp + webinar (canal de conversão mais importante da operação)
- Meta Ads + webinar (como estruturar tráfego pro evento)
- Números reais de CPA, show up, conversão de webinar (benchmarks)
- Bastidores de lançamento pago (o que acontece nos bastidores de uma operação de 7 dígitos)
- IA para roteiro de webinar, pitch, copy de anúncio
- Comparativos de checkout (Zouti, Hotmart, etc.)
- Como montar a equipe de um lançamento (gestor de tráfego, copy, editor)
